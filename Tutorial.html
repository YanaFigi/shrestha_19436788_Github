Tutorial for designing a web page.

There are a few step for developing a basic web page. They are on steps below:-

1. Make a Wireframe for your web page.

2. Imagine a design such as color combination for the web page.

3. Open a text editor like Sublime Text editor or Notepad++.

4. Save the file as .html so that the text editor captures it as a html file.

5. Type your code for you web page.

6. Click on the html file you saved as to review the web page you build.
